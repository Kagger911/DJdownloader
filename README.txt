## 🚀 Features

- 🎵 **Download from Spotify & YouTube:** Paste the link, hit download.
- 📝 **Batch Download:** Load a .txt list for bulk grabbing.
- 📁 **Smart Sorting:** Keeps Spotify & YouTube downloads tidy.
- 🌓 **Modern GUI:** Theme toggle, drag-n-drop, clipboard support.
- 🪄 **Automatic Setup:** Runs everything for you—no manual installs!
- 🖥️ **Windows-Ready:** (macOS/Linux soon)

---

## 🛠️ Requirements

- **Python 3.9+** (best with 3.11+)
- **Windows** (Tested)
- **Internet connection**

## ⚡ Installation & Usage

#### 1. Clone/download this repo

```bash
git clone https://github.com/Kagger911/DJdownloader.git
cd DJdownloader

2. Install dependencies the easy way:

python setup_djdownloader.py

Or, if you want to do it manually:

pip install -r requirements.txt

3. Launch the app

python downloader_app.py

4. Start downloading music!

    Paste a Spotify/YouTube link, or load a .txt of links.

    Select output folder/theme.

    Enjoy your tracks.

💡 Pro Tips

    Queue songs with drag-n-drop or copy-paste.

    Batch mode: Add a list of links via .txt.

    Match Mode explained:

        🎯 Exact (Default): Best match only

        ✅ Verified: Only download if exact, officially verified match

        🔄 Fuzzy: Looser matching for rare/obscure tracks
        (Hover over the blue “?” in the app for quick tips!)
