# SpotDL Ultra (DJDownloader)

A super simple all-in-one music downloader for DJs, streamers, and anyone who wants to grab songs from Spotify and YouTube.  
_Built with [SpotDL](https://github.com/spotDL/spotdl), [yt-dlp](https://github.com/yt-dlp/yt-dlp), and [FFmpeg](https://ffmpeg.org/)._

## Features
- Download from Spotify or YouTube â€” just paste the link!
- Smart GUI, drag-n-drop, and batch download support.
- Works on Windows (other OS coming soon).
- Automatic setup (just run `setup_djdownloader.py`).

## How to Use
1. Clone/download this repo.
2. Run `setup_djdownloader.py` (installs dependencies for you!).
3. Run `downloader_app.py` and start grabbing tracks.
